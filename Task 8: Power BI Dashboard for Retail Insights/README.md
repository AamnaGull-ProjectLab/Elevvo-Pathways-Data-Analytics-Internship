This folder contain my interactive Power BI dashboard along with the Superstore Sales Dataset by Kaggle. In this task i have covered the topics of data modeling, interactive visualization and dashboarding.

What this Dashboard shows:
1. KPIs such as Total Revenue, Top Products etc
2. Map showing states with their orders and sales
3. Sales  by region, category, shipment class, sub categories
4. Total Orders, Average Sales per order
5. Bar charts for top selling products
6. Countries with the highest Sales
   
Tools and libraries used:
Power BI Desktop (Added slicers, filters, maps, bar charts, pie charts )
