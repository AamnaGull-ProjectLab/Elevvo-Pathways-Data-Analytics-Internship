# Walmart Retail Sales Time Series Analysis

This project analyzes Walmart retail sales data to identify trends, seasonal patterns, and revenue distribution across stores and product departments.

## Project Objective

The objective of this project is to perform time series analysis on retail sales data and generate insights about sales performance over time.

## Dataset

The dataset used in this project is the **Walmart Sales Forecasting dataset** from Kaggle.

It includes the following main variables:

- Store: Store number representing different locations
- Dept: Department representing product categories
- Date: Weekly sales date
- Weekly_Sales: Recorded weekly revenue
- IsHoliday: Indicates whether the week includes a holiday

## Tools and Technologies

- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed

The following analyses were performed:

- Monthly sales trend visualization
- Moving average analysis
- Seasonal sales pattern analysis
- Revenue breakdown by store (region)
- Revenue breakdown by department (product category)
- Holiday vs non-holiday sales comparison

## Key Insights

- Retail sales show noticeable seasonal patterns across different months.
- Some stores consistently generate higher revenue than others.
- Certain departments contribute significantly to total sales.
- Sales during holiday weeks tend to be higher compared to regular weeks.
